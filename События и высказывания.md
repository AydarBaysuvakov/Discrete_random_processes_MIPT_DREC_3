| События                  | Высказывания        |
| ------------------------ | ------------------- |
| достоверное событие      | верное (1)          |
| невозможное событие      | ложное (0)          |
| дополнительное к событию | отрицание ($\neg$)  |
| пересечение событий      | конъюкция ($\land$) |
| объединение событий      | дизъюнкция ($\lor$) |
Любое событие можно описать разными высказываниями, поэтому говорят, что между высказываниями и событиями есть [[сюрьекция]].
