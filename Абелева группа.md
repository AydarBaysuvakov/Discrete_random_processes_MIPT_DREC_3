[[Полугруппа, абелев моноид|Абелев моноид]] $(A, +, 0)$ называется **абелевой группой**, если $\forall a \in A \exists b \in A: a + b = b + a = 0$.

*Обозначения*.  $b = -a, \space x + (-y) = x - y$.

*Комментарий*. В абелевой группе $\forall a \in A \exists ! b = -a$.
(Если $b_1, b_2$ — обратные к $a$, то $b_1 = b_1 + 0 = b_1 + (a + b_2) = (b_1 + a) + b_2 = 0 + b_2 = b_2$)
