Временной ряд (статистика измерений):

| $N$ измерения  | 1       | 2       | $\cdots$ | $N_1$      | $\cdots$ | $N_2$      | $\cdots$ | $N_k$      | $\cdots$ |     |
| -------------- | ------- | ------- | -------- | ---------- | -------- | ---------- | -------- | ---------- | -------- | --- |
| Значение $\xi$ | $x_1^Э$ | $x_2^Э$ |          | $x_{N1}^Э$ |          | $x_{N2}^Э$ |          | $x_{Nk}^Э$ |          |     |
$$F_{Э_\xi}(x) = \frac{1}{N_k}|\{n\in\{1, \cdots, N_k\}|_: x_n^Э<x\}|$$
$\xi$ — [[случайная величина]], которая измеряется каким-либо прибором.
$Э$ — эксперимент, в процессе которого проводится измерение случайной величины $\xi$.
$k$ — номер итерации группы измерений.

Идея следующая: $F_{Э_\xi}(x)$ сходится при $k \rightarrow \infty$ — необходимое условие для нашей теории. Как правило, это условие выполняется.

Масштабируем эксперимент, добавим новые приборы, измеряющие какие-нибудь другие случайные параметры.

| $N$ измерения | 1           | 2           | $\cdots$    | $N$         |
| ------------- | ----------- | ----------- | ----------- | ----------- |
| $\xi_1$       | $x_{1,1}^Э$ | $x_{1,2}^Э$ | $\vdots$    | $x_{1,N}^Э$ |
| $\xi_2$       | $x_{2,1}^Э$ | $x_{2,2}^Э$ | $\vdots$    | $x_{2,N}^Э$ |
| $\vdots$      | $\cdots$    | $\cdots$    | $x_{k,n}^Э$ | $\cdots$    |
| $\xi_M$       | $x_{M,1}^Э$ | $x_{M,2}^Э$ | $\vdots$    | $x_{M,N}^Э$ |

$$\vec{x} \in \mathbb R^M, \space \vec{x} \rightarrow F_\vec{\xi}(\vec{x}) = \frac{1}{N}|\{n\in\{1, \cdots, N\}|_: \forall k = 1\cdots M:x_{k,n}^Э<x_k\}|$$
