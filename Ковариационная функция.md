$\xi = (\xi_n)_{n\in \mathbb Z}$ — стационарный в широком смысле случайный процесс
$R_\xi: \mathbb Z\ni n \rightarrow R_\xi(n) = Cov(\xi_n, \xi_0)$ — ковариационная функция.

### Следствия
1. $Cov(\xi_n, \xi_m) = R_\xi(n-m)$
2. $R_\xi(0) = Cov(\xi_0, \xi_0) = R_\xi(n-n) = Cov(\xi_n, \xi_n) = \mathbb D(\xi_n) = \|\xi_n\|^2 \ge 0$
3. $R_\xi(0) = 0 \Rightarrow \|\xi_n\|=0$ — тривиальность процесса.
4. $|Cov(\xi_n, \xi_m)| \le \|\xi_n\|\|\xi_m\| = R_\xi(0)$
5. $R_\xi^*(n) = R_\xi(-n)$.
